🧠 My AI/ML Lab Setup on Ubuntu

Posted on July 7, 2025

I recently repurposed an old laptop into a dedicated AI/ML lab running Ubuntu Server. This setup gives me full control over my development environment while remaining lightweight and secure.

🔧 Hardware Used

  • Old Dell Latitude laptop
  • Intel Core i5 processor
  • 8 GB RAM
  • 500 GB SSD

💻 OS Installation

I installed Ubuntu Server instead of Ubuntu Desktop to keep the system lean. The installer is straightforward, and I opted for OpenSSH during setup so I could manage it remotely.

📦 Package Setup with Conda

I installed Miniconda for Python package management. Here's the basic flow:

sudo apt update
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh

After installation, I created an ML-focused environment:

conda create -n ml-lab python=3.10 scikit-learn pandas jupyterlab matplotlib

🌐 Remote Access with Cloudflare Tunnels

I wanted secure access to the server without exposing it directly over the internet. I configured a Cloudflare Tunnel and mapped subdomains like:

  • jupyter.mvsastry.dev → JupyterLab interface
  • docs.mvsastry.dev → Flask-powered documentation

🚀 Final Thoughts

This setup gives me flexibility to run experiments from anywhere, share live demos, and manage everything through VS Code, SSH, or the browser. The next step is automating backups and deploying web-facing ML APIs — stay tuned!
